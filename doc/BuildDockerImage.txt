* How to image file

This is a memo for making a docker image file for distribution.

Goto docker subdirectory
Make sure that PetraM repository is updated/cleaned.

docker build --no-cache -t twopi/base -f Dockerfile.base .
docker build --no-cache -t twopi/pymfem -f Dockerfile.pymfem .
docker build --no-cache -t twopi/petram -f Dockerfile.petram .

(?) docker build --squash -t twopi/petram -f Dockerfile.petram .

# then, save the image to file and place it to a download site
docker save -o twopi_petram_image twopi/petram
